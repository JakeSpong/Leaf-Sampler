🔥 Connect Firebase

For shared grid data across all users and devices, connect a free Firebase Firestore database. Your pin codes and coordinates are stored privately on your device — only the grid cell counts and anonymised pin records are shared.

1. Go to console.firebase.google.com → Create a project (free Spark plan)
2. Add a Web app → copy the firebaseConfig object
3. Go to Firestore Database → Create database → Start in test mode
